# 寻印 (Xunyin) - 产品概述

## 产品定位

寻印是一款文化探索类应用，包含三个主要组成部分：

1. **管理后台 (Admin)** - 企业级权限管理系统，用于管理城市、文化之旅、印记等内容
2. **移动端 App (Flutter)** - 用户端应用，提供文化之旅探索、印记收集、AR 互动等功能
3. **后端服务 (NestJS)** - 统一的 API 服务，同时支持 Admin 和 App 端

## 核心业务概念

| 概念 | 说明 |
|------|------|
| 城市 (City) | 可探索的城市，包含多条文化之旅 |
| 文化之旅 (Journey) | 城市内的主题探索路线，包含多个探索点 |
| 探索点 (ExplorationPoint) | 路线上的打卡点，支持手势识别、拍照、寻宝等任务 |
| 印记 (Seal) | 完成探索后获得的数字徽章，支持区块链存证 |
| 用户进度 (JourneyProgress) | 用户的文化之旅完成进度 |

## 用户体系

- **Admin 用户 (SysUser)** - 管理后台用户，基于 RBAC 权限模型
- **App 用户 (AppUser)** - 移动端用户，支持微信/邮箱/Google/Apple 登录

## 特色功能

- 区块链印记存证（支持多链：蚂蚁链、至信链、BSN、Polygon 等）
- AR 互动体验
- 背景音乐系统
- 积分与等级体系

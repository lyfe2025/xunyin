## 使用方法

### 1. 导入组件
```vue
import TablePagination from '@/components/common/TablePagination.vue'
```

### 2. 替换原有的分页显示
**替换前:**
```vue
<div class="flex justify-end">
  <div class="text-sm text-muted-foreground p-2">
    共 {{ total }} 条
  </div>
</div>
```

**替换后:**
```vue
<TablePagination
  v-model:page-num="queryParams.pageNum"
  v-model:page-size="queryParams.pageSize"
  :total="total"
  @change="getList"
/>
```

### 3. 确保数据结构正确
确保组件中有以下数据:
```typescript
const queryParams = reactive({
  pageNum: 1,
  pageSize: 10,
  // ... 其他查询参数
})

const total = ref(0)
```

## 组件特性
- 显示记录范围: "显示第 1 - 20 条，共 602 条记录"
- 每页条数选择: 10/20/50/100
- 简洁的分页按钮: 上一页 / 第 X/Y 页 / 下一页
- 自动禁用边界按钮
- 响应式设计

## 注意事项
1. 组件使用 `v-model` 双向绑定 `pageNum` 和 `pageSize`
2. 切换每页条数时会自动重置到第一页
3. `@change` 事件会在页码或每页条数改变时触发
4. 确保 `getList` 方法存在并能正确加载数据

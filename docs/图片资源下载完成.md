## 使用方式

### 重新下载图片

如果需要重新下载所有图片：

```bash
bash server-nestjs/scripts/download-all-images.sh
```

### 导入初始数据

图片下载完成后，可以运行 seed 导入初始数据：

```bash
pnpm --filter server-nestjs db:seed
```

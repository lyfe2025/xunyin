## 💡 文档使用建议

### 对于新人
1. 按照 [快速开始](./指南/快速开始.md) 搭建环境
2. 阅读 [pnpm使用指南](./指南/pnpm使用指南.md) 了解命令
3. 阅读 [Prisma使用指南](./指南/Prisma使用指南.md) 初始化数据库
4. 开发时参考 **开发规范** 目录

### 对于老手
1. 直接使用 **快速导航** 查找需要的内容
2. 需要了解规范时查看 **开发规范** 目录
3. 需要回顾历史时查看 **历史记录** 目录

### 对于维护者
1. 新增功能后更新对应的 **指南** 文档
2. 制定新规范后添加到 **开发规范** 目录
3. 完成重大优化后记录到 **历史记录** 目录
